# GitAuto

## Installation

Add this line to your application's Gemfile:

    gem 'git-auto'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install git-auto

## Usage

* cleaned
* reordered
* commented
* fixed-typo
* changed-wording
* downsized

* moved files
* removed files
* added files

Pass multiple actions:

    $ git auto cleaned reordered
